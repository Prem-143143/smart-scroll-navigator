# **Privacy Policy for Smart Scroll Navigator**

**Last updated:** February 2026

Smart Scroll Navigator ("the extension") is designed to enhance the user experience on ChatGPT by providing navigation and interface improvements. Your privacy is important to us, and this Privacy Policy explains how information is handled.

## **1\. Information We Collect**

The extension does **not collect, store, transmit, or share any personally identifiable information** or sensitive user data.

Specifically, we do not collect:

* Personal information (such as name, email address, or contact details)  
* Authentication information (such as passwords or credentials)  
* Financial or payment information  
* Health information  
* Location data  
* Personal communications  
* Browsing history outside the functionality required for the extension  
* Website content for external storage or transmission

## **2\. Local Storage**

The extension may store user preferences locally within the browser using Chrome’s storage API. This information is stored only on the user’s device and is not transmitted to any external servers.

Examples include:

* Interface preferences  
* Navigation settings  
* Feature toggles

## **3\. How We Use Permissions**

The extension requests certain permissions solely to provide its core functionality:

* **storage** – to save user preferences locally.  
* **Host permissions (chat.openai.com)** – to operate only on the ChatGPT website where the extension features are applied.

These permissions are used only for the stated functionality and not for data collection.

## **4\. Data Sharing**

We do **not sell, trade, or share** any user data with third parties because no user data is collected.

## **5\. Third-Party Services**

The extension does not use external analytics, advertising services, or remote code that would collect user information.

## **6\. Changes to This Policy**

This Privacy Policy may be updated in the future if features change. Any updates will be reflected with a new “Last updated” date.
